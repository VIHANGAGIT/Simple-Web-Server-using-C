A Simple Web Server using C

This is a web server created using C programming language. It incorporates several features including robust error handling, concurrent connections handling, and support for various types of files such as HTML, CSS, serveral image formats, and more.

GitHub Repo

Features

Basic HTTP GET Handling
HTTP Response Handling
MIME Type identification
Handling multiple client requests by forking

Supported File Types

Text: .htm, .html
Images: .gif, .jpg, .jpeg, .png, .ico
Stylesheets: .css
Archives: .zip, .gz, .tar
